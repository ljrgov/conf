# 规则源文件填写指南

## 通用说明

- 每行一条规则
- `#` 或 `;` 开头为注释
- 策略部分（`DIRECT` / `PROXY` / `REJECT` 等）可写可不写，处理时会自动剥离
- `no-resolve` 可写可不写，处理时会自动剥离

---

## 域名

| 输入示例 | 说明 |
|---------|------|
| `DOMAIN,ad.com` | 完整域名 |
| `DOMAIN-SUFFIX,google.com` | 域名后缀（含子域名） |
| `DOMAIN-KEYWORD,google` | 域名关键词 |
| `DOMAIN-WILDCARD,*.google.com` | 通配符（`*` 多字符，`?` 单字符）|
| `DOMAIN-REGEX,^ad\..*\.com$` | 正则表达式 |
| `GEOSITE,youtube` | GeoSite 数据库 |

**裸值自动识别：**

| 输入示例 | 自动转换 |
|---------|---------|
| `ad.com` | `DOMAIN,ad.com` |
| `.google.com` | `DOMAIN-SUFFIX,google.com` |
| `*google*` | `DOMAIN-WILDCARD,*google*` |
| `google`（无点）| `DOMAIN-KEYWORD,google` |

---

## URL 路径

| 输入示例 | 自动转换 | 说明 |
|---------|---------|------|
| `/igd.xml` | `URL-REGEX,/igd\.xml` | 数字自动转 `\d+`，`.` 自动转义 |
| `/123/resolve` | `URL-REGEX,/\d+/resolve` | |
| `https://ad.com` | `DOMAIN,ad.com` | 仅提取域名 |
| `https://ad.com/path` | `DOMAIN,ad.com` + `URL-REGEX,/path` | 域名和路径分别生成 |
| `https://*.ad.com/path` | `DOMAIN-WILDCARD,*.ad.com` + `URL-REGEX,/path` | |

> ⚠️ URL-REGEX 仅 Surge 输出，Mihomo / sing-box 不支持

## IP

| 输入示例 | 说明 |
|---------|------|
| `IP-CIDR,127.0.0.0/8` | IPv4 段 |
| `IP-CIDR6,::1/128` | IPv6 段 |
| `IP-SUFFIX,8.8.8.8/24` | IP 后缀匹配 |
| `IP-ASN,13335` | 目标 ASN |
| `GEOIP,CN` | GeoIP 国家代码 |

**裸值自动识别：**

| 输入示例 | 自动转换 |
|---------|---------|
| `127.0.0.1` | `IP-CIDR,127.0.0.1/32` |
| `127.0.0.0/8` | `IP-CIDR,127.0.0.0/8` |
| `::1` | `IP-CIDR6,::1/128` |

| 输入示例 | 说明 |

| `SRC-IP-SUFFIX,192.168.1.201/8` | 来源 IP 后缀 |
| `SRC-IP-ASN,9808` | 来源 ASN |
| `SRC-GEOIP,cn` | 来源 GeoIP |

## 端口

| 输入示例 | 说明 |
|---------|------|
| `DST-PORT,80` | 目标端口（单个）|
| `DST-PORT,80,443,8080` | 目标端口（多个）|
| `DST-PORT,8000-9000` | 目标端口（范围）|
| `SRC-PORT,7777` | 来源端口 |

**裸值自动识别：**

| 输入示例 | 自动转换 |
|---------|---------|
| `80` | `DST-PORT,80` |
| `8000-9000` | `DST-PORT,8000-9000` |
| `80,443` | `DST-PORT,80,443` |

> `DEST-PORT` / `PORT` 写法也支持，自动转换为 `DST-PORT`

---

## 入站

| 输入示例 | 说明 |
|---------|------|
| `IN-PORT,7890` | 入站监听端口 |
| `IN-TYPE,SOCKS/HTTP` | 入站协议类型 |
| `IN-USER,mihomo` | 入站用户名 |
| `IN-NAME,ss` | 入站名称 |

> ⚠️ 仅 Mihomo 支持，Surge 仅支持 `IN-PORT`，sing-box 不支持

## 进程

| 输入示例 | 说明 |
|---------|------|
| `PROCESS-NAME,curl` | 进程名 |
| `PROCESS-NAME,chrome.exe` | Windows 进程名 |
| `PROCESS-NAME,com.google.android.youtube` | Android 包名 |
| `PROCESS-NAME-WILDCARD,*telegram*` | 进程名通配符 |
| `PROCESS-NAME-REGEX,curl$` | 进程名正则 |
| `PROCESS-PATH,/usr/bin/curl` | 进程完整路径（Linux/Mac）|
| `PROCESS-PATH,C:\Program Files\curl.exe` | 进程完整路径（Windows）|
| `PROCESS-PATH-WILDCARD,/usr/*/curl` | 进程路径通配符 |
| `PROCESS-PATH-REGEX,.*bin/curl` | 进程路径正则 |

**`PROCESS,` 简写自动识别：**

| 输入示例 | 自动转换 |
|---------|---------|
| `PROCESS,curl` | `PROCESS-NAME,curl` |
| `PROCESS,/usr/bin/curl` | `PROCESS-PATH,/usr/bin/curl` |
| `PROCESS,C:\xxx\chrome.exe` | `PROCESS-PATH,C:\xxx\chrome.exe` |
| `PROCESS,*telegram*` | `PROCESS-NAME-WILDCARD,*telegram*` |

## Adblock 语法

| 输入示例 | 处理方式 |
|---------|---------|
| `adblock.turtlecute.org##.adbox` | 直接丢弃（元素隐藏规则）|
| `/pagead.js$domain=ad.com` | 提取 `DOMAIN,ad.com` + `URL-REGEX,/pagead\.js` |
| `/widget/ads.` | `URL-REGEX,/widget/ads\.` |
