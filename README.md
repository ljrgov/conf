# 🚀 Rules and config 

本项目是一个自动化的网络代理规则同步库，旨在通过脚本实现规则的“精准克隆”与“个性化微调”。

---

### 🛠️ 处理逻辑 (Automated Logic)

为了适应个人网络环境，本项目通过 GitHub Actions 对原始规则执行了以下操作：

* **自动同步**：每日定时从上游仓库拉取最新的 `.list` 文件。
* **格式转换**：统一输出为 `.txt` 纯文本格式，适配多种代理工具。
* **精准过滤 (Exclude)**：通过规则映射表，自动剔除原规则中可能导致分流回环或冲突的域名。
* **按需补全 (Include)**：手动维护补全列表，追加缺失的子域名以确保规则完整性。
